abrir um repositório
git clone url

CRIAR NOVA BRANCH
git checkout -b [nome-branch]

NAVEGAR ENTRE BRANCHES
git checkout [nome-branch]

ATUALIZAR BRANCH (puxar os arquivos atualizados do repositório)
git pull

ADICIONAR TODOS OS ARQUIVOS PARA GITHUB
git add . 
(se quiser um arquivo em específico mudar para: Victor/Alg/nomeDoArquivo)

ADICIONAR MENSAGEM AO SEU COMMIT
git commit -m '[mensagem]'

ENVIA MUDANÇAS PARA O GITHUB
git push (se houver mais de uma branch ( origin [nome-branch] ))
